    list

### Box Add

    vagrant box add ADDRESS

##### example

    vagrant box add ubuntu/trusty32

### Box List

    vagrant box list

### Box outdated

    vagrant box outdated

### Box Remove

    vagrant box remove NAME

### Box Package

    vagrant package --output mynew.box

### Box update

    vagrant box update
